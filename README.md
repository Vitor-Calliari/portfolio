# Portfólio — Vitor Bernhard Calliari

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, com tema escuro/claro, animações de scroll e partículas interativas no fundo.

🔗 **[Acesse o portfólio](https://vitor-calliari.github.io/portfolio)**

---

## Funcionalidades

- **Tema escuro/claro** — alternável via botão no menu de navegação
- **Grade de partículas interativas** — animação em canvas que reage ao movimento do mouse
- **Scroll reveal** — elementos surgem suavemente ao rolar a página
- **Contador animado** — números nas estatísticas incrementam ao entrar na viewport
- **Efeito de digitação** — terminal simulado com comandos sendo digitados
- **Design responsivo** — adaptado para desktop e mobile
- **Navegação fixa** com backdrop blur ao rolar

## Deploy

O portfólio é hospedado via **GitHub Pages**. Qualquer `push` na branch `main` atualiza o site automaticamente em até 2 minutos.

Para acompanhar o status do deploy, acesse a aba **Actions** do repositório.
